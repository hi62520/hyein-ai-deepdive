# 뤼튼(Wrtn) — 외로움 경제를 먹는 한국 AI 포털의 15배 성장

## 누구/무엇인가

뤼튼테크놀로지스(Wrtn Technologies)는 2021년 4월 서울에서 창업한 한국 AI 플랫폼 기업입니다. CEO 이세영(Seyoung Lee)은 고등학교 2학년 때 청소년 학술대회(KSCY)를 직접 설립한 뒤, 코로나19로 행사가 온라인으로 전환되면서 10대들의 글쓰기 실력이 생각보다 훨씬 낮다는 사실을 발견했습니다. 이 불편함이 뤼튼 창업의 씨앗이 되었습니다. 6명이 시작한 스타트업은 2026년 4월 기준 월간활성사용자(MAU) 527만 명을 보유하며, 연 환산 매출(ARR) $100M 돌파를 목전에 두고 있습니다. 누적 투자 유치액은 약 1,300억원이며, 2025년 3월 시리즈B 라운드에서만 1,080억원을 조달했습니다. Forbes Korea 30세 미만 30인에 두 번 연속 이름을 올린 이세영 대표는 "2030년 아시아 최대 하이퍼스케일러"를 공개 목표로 내걸었습니다.

---

## 무엇이 특별한가

**1. 외로움 경제(Loneliness Economy)를 AI 엔터테인먼트로 풀었습니다**

뤼튼의 핵심 수익 엔진은 AI 스토리텔링 플랫폼 '크랙(Crack)'입니다. 한국 서비스명은 크랙, 일본 서비스명은 캬라푸(Kyarapu)인데, 일반 캐릭터 챗봇과 결정적으로 다른 점은 사용자가 단순히 캐릭터와 대화하는 것이 아니라 자신이 주인공이 되는 인터랙티브 내러티브 안에 들어간다는 것입니다. 이세영 대표는 이 서비스를 두고 "pure dopamine…somewhere between a story, a live experience, and a gaming experience(순수한 도파민이다... 소설, 라이브 체험, 게임 경험의 어딘가에 존재한다)"라고 직접 표현했습니다. 평균 사용자가 하루 2시간을 씁니다. 넷플릭스, 유튜브와 비슷한 수준입니다. 외로움 문제가 갈수록 심각해지는 MZ 세대 시장을 정조준한 결과입니다.

**2. 무료 포털 → 유료 엔터테인먼트로의 피벗이 15배 성장의 비밀입니다**

2024년까지 뤼튼은 GPT·Claude·Gemini 등 최고 성능 모델을 무료로 제공하는 AI 포털 전략을 취했습니다. MAU는 빠르게 성장했지만 외부 LLM API 비용 부담으로 영업손실이 연 260억~290억원에 달했으며 매출은 고작 30억원 수준이었습니다. 그런데 2025년 크랙의 유료화를 시작하자 한 달 만에 월 매출 10억원을 돌파했습니다. 유료 사용자 리텐션이 70% 이상입니다. 이 전환이 2025년 연매출 471억원(전년 대비 15배)과 2025년 말 ARR $70M이라는 숫자로 이어졌습니다. Fortune은 2026년 3월 "뤼튼이 $100M ARR 돌파를 향해 가고 있다"는 기사를 냈습니다.

**3. 한→일→미 글로벌 확장 순서가 전략적으로 탁월합니다**

뤼튼은 한국 시장에서 검증한 뒤 2025년 중반 일본에 '캬라푸(Kyarapu)'를 출시했습니다. 일본은 캐릭터 IP·소설·만화 문화가 깊어 AI 스토리텔링 서비스의 정서적 수용성이 매우 높습니다. Fortune은 뤼튼이 한국과 일본을 합쳐 MAU 500만명 이상을 확보했다고 보도했습니다. 그리고 2026년 중반 미국 시장 진출을 공식화했습니다. 영어권보다 훨씬 동질적인 문화권에서 제품-시장 적합성을 완성한 뒤 미국이라는 큰 시장을 노리는 순서입니다. IPO는 2028년을 잠정 목표로 삼고 있으며, 2027년 ARR $700M(2025년 대비 10배) 달성이 전제 조건입니다.

**4. B2C 무료 포털 + B2C 유료 엔터테인먼트 + B2B AX, 세 축 동시 운영**

대부분의 한국 AI 스타트업이 한 방향만 보는 것과 달리, 뤼튼은 세 축을 동시에 운영합니다. 무료 포털(wrtn.ai)은 MAU 트래픽 확보와 브랜드 인지도, 크랙은 B2C 수익화, 그리고 2026년 들어 공개한 AX(AI Transformation) 서비스는 B2B 기업 AI 전환 수주입니다. "B2C에서 B2B까지 '일하는 AI' 기준을 제시하겠다"는 이세영 대표의 발언은 단순한 수사가 아니라 세 축 수익화의 현실 반영입니다.

## 와당탕/느린호밀 적용 포인트

**오늘 30분 (micro)**: 옵시디언에서 "외로움 경제" 노트를 새로 만들어 뤼튼 크랙 전략(하루 2시간 사용·리텐션 70%)을 비서앱(hyein-daily)이나 wadangtang-erp에 적용할 수 있는 아이디어 3가지 적어두기. 예: 비서앱의 아침 루틴 스트릭 기능, 와당탕 고객 관계 관리에 스토리텔링 요소 추가, 느린호밀 뉴스레터에 독자 참여형 서사 삽입 등.

**이번 주 1-2시간 (mid)**: 뤼튼의 무료 포털 전략을 비서앱에 벤치마킹합니다. 비서앱 내에 "AI 모델 멀티 접속" 뷰를 하나 만들어 Claude API와 Gemini API 응답을 나란히 보여주는 간단한 비교 UI를 구현해보세요. 라이브러리는 기존 Anthropic SDK + Google Generative AI SDK를 사용합니다.

```python
import anthropic
import google.generativeai as genai

async def compare_models(prompt: str):
    # Claude 응답
    client = anthropic.Anthropic()
    claude_resp = client.messages.create(
        model="claude-sonnet-4-6",
        max_tokens=500,
        messages=[{"role": "user", "content": prompt}]
    )
    
    # Gemini 응답 (API 키 필요)
    genai.configure(api_key="YOUR_KEY")
    model = genai.GenerativeModel("gemini-2.0-flash")
    gemini_resp = model.generate_content(prompt)
    
    return {
        "claude": claude_resp.content[0].text,
        "gemini": gemini_resp.text
    }
```

**이번 달 실험 (macro)**: 느린호밀 뉴스레터에 "독자 참여형 스토리 실험"을 한 번 붙여봅니다. 뤼튼 크랙처럼 독자가 선택지를 고르는 인터랙티브 뉴스레터를 1회 발행하고, 오픈율·클릭율·구독 해지율을 기존 호와 비교합니다. 성공 기준: 클릭율 기존 평균 대비 +50% 이상. 측정 도구: Stibee 또는 Mailchimp 리포트.

## 한국 솔로 운영자 맥락에서 주의

**GPT 래퍼 사업의 비용 구조를 과소평가하면 안 됩니다.** 뤼튼조차 2024년까지 외부 LLM API 비용으로 연 260억원 이상의 영업손실을 냈습니다. 뤼튼은 투자금 1,300억원이 있어서 버텼지만, 1인 운영자가 비슷한 구조로 서비스를 만들면 API 비용 부담이 치명적입니다. 대표님이 개발 중인 비서앱과 ERP는 '내부 도구'로 활용하는 구조라 다르지만, 만약 외부 사용자에게 AI API를 무료로 제공하는 방향을 고려한다면 토큰 사용 상한(rate limit)과 수익화 시점을 반드시 선계획해야 합니다.

**'외로움 경제' 콘텐츠는 한국 규제 리스크가 있습니다.** 뤼튼의 크랙은 AI 캐릭터 챗, 인터랙티브 소설 장르인 만큼 특히 청소년 이용자 관련 내용 규제, 성인 콘텐츠 분류 기준, 개인정보 보호법 이슈가 복잡하게 얽혀 있습니다. 나무위키에 "크랙 논란 및 사건 사고" 별도 문서가 존재할 만큼 실제 운영 중 사건이 있었습니다. 느린호밀 뉴스레터나 와당탕 연구소의 커뮤니티 서비스에 유사한 기능을 추가할 경우, 콘텐츠 가이드라인과 연령 확인 절차를 미리 설계하세요.

## 강의 메모 후보 (Pain/숫자/삽질/훅)

**Pain**: 이세영 대표는 코로나 시기에 청소년 학술대회를 온라인으로 전환하다가, 10대들이 생각을 글로 풀어내는 능력이 예상보다 훨씬 낮다는 사실을 발견했습니다. 학생들은 글 쓰는 것 자체를 두려워했고, 이 불편함을 AI로 해소하겠다는 생각이 뤼튼의 시작이었습니다. 창업 당시 이세영 대표는 스물한 살이었습니다.

**숫자**: 2024년 매출 30억원, 2025년 매출 471억원. 1년 만에 정확히 15.7배 성장입니다. 같은 기간 적자를 내면서도 버틸 수 있었던 건 시리즈B 1,080억원 투자금이 있었기 때문입니다. 비교: Character.AI의 한국 경쟁사 지위였던 뤼튼이 이제 글로벌에서 Character.AI와 직접 경쟁하는 위치가 되었습니다.

**삽질**: 뤼튼은 2024년까지 "무조건 무료"라는 전략 하나로 달렸습니다. GPT-4, Claude, Gemini를 전부 무료로 쓸 수 있게 했고 MAU는 쌓였지만, 외부 LLM API 비용과 인건비·인프라를 합산하면 연간 영업손실이 260억~290억원에 달했습니다. 매출 30억원에 손실 280억원이면 사실상 투자금으로 사용자를 사고 있는 구조였습니다. 크랙 유료화 이후 한 달 만에 월 매출 10억원이 나온 것을 보면, 진작 수익화를 시도했어야 했습니다.

**훅**: "하루 2시간 동안 앱을 쓰게 만드는 AI 서비스를 한국 스타트업이 만들었습니다. 넷플릭스, 유튜브와 같은 시간입니다. 그 비결은 더 좋은 AI 모델이 아니라, 외로움이었습니다." 청중이 고개를 갸우뚱할 만한 한 문장입니다.
