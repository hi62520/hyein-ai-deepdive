# 즐겨찾기 Deep Dive

> deep dive 파일을 열어 frontmatter의 `starred: false` → `true`로 바꾸면 아래 표에 자동 추가됩니다.

```dataview
TABLE WITHOUT ID
  file.link AS "Deep Dive",
  category AS "카테고리",
  date AS "날짜"
FROM "04 INQUIRY/AI탐구"
WHERE starred = true
SORT date DESC
```

## 카테고리별

```dataview
TABLE WITHOUT ID
  file.link AS "Deep Dive",
  date AS "날짜"
FROM "04 INQUIRY/AI탐구"
WHERE starred = true
GROUP BY category
SORT category ASC
```
