# Pieter Levels — 혼자서 $3.5M ARR, 비결은 '3시간 빌드'

## 누구인가

네덜란드 출신 솔로 개발자 Pieter Levels(@levelsio). 직원 없이 NomadList(2014), RemoteOK, PhotoAI, InteriorAI, fly.pieter.com 등 40개 이상의 프로덕트를 혼자 운영. 총 ARR $3.5M+, 순이익률 90% 이상. X(트위터) 팔로워 600K+.

## 무엇이 특별한가

### fly.pieter.com — 17일 만에 $1M ARR

2025년 3월, Cursor + Claude 3.7 Sonnet + Grok 3를 이용해 **3시간 만에** 브라우저 기반 멀티플레이어 비행 시뮬레이터를 제작. 게임 개발 경험 제로였다.

- 런칭 10일: $38,360/mo (블림프·F-16 인게임 광고판 판매)
- 런칭 17일: **$87,000 MRR → $1M ARR 돌파**
- 누적 비행자: 320,000명+

> "✨ https://fly.pieter.com has now gone from $0 to $1 million ARR in just 17 days!"
> — @levelsio 트윗, 2025년 3월

수익 모델은 단순하다. 게임 내 광고판(블림프) + F-16 업그레이드($29.99) + 인게임 광고. 구독 없이 일회성 결제로 MRR을 만들었다.

### PhotoAI — 18개월 $0 → $132K MRR

2022년 말 Stable Diffusion을 보고 즉시 실험 시작. AI 프로필 사진 생성 서비스를 만들어 18개월 만에 $132K MRR(연간 $1.65M ARR) 달성. 전체 수익의 약 70%를 차지.

### '바이브 코딩(Vibe Coding)' 철학

Pieter가 직접 명명한 개발 방식. AI 도구(Cursor, Claude, Grok)에게 원하는 것을 말로 설명하고, AI가 생성한 코드를 방향만 잡아주며 반복 개선. **기획서도 없고, 팀도 없고, VC도 없다.**

2025년에는 직접 $40,000 상금 규모의 'Vibe Coding Game Jam'을 주최했다(Cursor, Bolt 후원). 조건: 코드의 80% 이상을 AI가 작성할 것.

> "I don't raise money, I don't have a team. I need to go fast to see if an idea works."

핵심 도구 스택: **Cursor(주개발) + Claude 3.7 Sonnet + Grok 3 + Three.js + Stripe**

### 10년짜리 배포 전략

숫자를 공개 트윗한다. 매출, 실패, 코드 스니펫 모두. 600K 팔로워는 그냥 쌓인 게 아니라 10년 빌드인퍼블릭의 결과물이다. PhotoAI가 18개월 만에 $132K MRR에 도달한 건 제품력도 있지만 런칭 당일 600K 팔로워에게 닿는 '무료 배포망'이 핵심이었다.

## 와당탕/느린호밀 적용 포인트

1. **fly.pieter.com 수익 모델을 ERP 대시보드에 적용**: 인게임 광고판처럼 '이름 올리기' 방식으로 느린호밀 고객사에게 ERP 내 배너/메모 공간을 유료 제공하는 아이디어 테스트 — 이번 주 hyein-daily 비서앱에 "협찬 메모" 기능 1개 추가해보기.

2. **3시간 빌드 챌린지**: Cursor + Claude로 와당탕 ERP의 신규 모듈 하나(예: 재고 알림)를 "기획 없이 바로 프롬프트로" 만드는 실험을 이번 주 내로 1회 시도. 결과를 X나 블로그에 빌드인퍼블릭으로 공개.

3. **매출 공개 루틴 시작**: Pieter처럼 월 1회 MRR 공개 트윗 → 느린호밀·와당탕 각각의 수치를 투명하게 공유. 팔로워 수가 작아도 시작이 중요.
