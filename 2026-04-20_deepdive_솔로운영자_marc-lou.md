# Marc Lou — "일단 출시"로 연 10억 번 프랑스 1인 개발자

## 누구인가

Marc Lou(마크 루, 본명 Marc Louvion)는 프랑스 파리 출신의 1인 개발자·소프레너입니다. 2016년 영화 <소셜 네트워크>를 보고 창업에 매료돼 독학으로 코딩을 시작했고, 서울에서 AI 스타트업을 공동 창업했다가 실패하는 등 4년간 처절히 고전했습니다. 한때 $20,000(약 2,700만 원)밖에 남지 않은 상황에서 발리로 이주해 "1주일에 1제품 출시" 전략으로 전환했습니다. 2023년 9월 ShipFast를 론칭해 첫 달에 $40,000을 벌었고, 2025년 총매출 $1,032,000(약 14억 원)을 달성했습니다. 2026년 1월에는 13개 제품 포트폴리오에서 월 $94,800을 벌어들이고 있으며, 직원은 0명입니다. X(구 트위터) 팔로워는 250,000명이 넘습니다.

## 무엇이 특별한가

**1. "완성이 아니라 출시가 곧 제품"이라는 철학**

Marc Lou의 핵심 원칙은 하나입니다. "Ship the 24-hour version of your idea(아이디어의 24시간 버전을 출시하라)." 그는 ShipFast를 설명할 때 "내가 27개 프로젝트에서 재사용하던 코드를 정리한 것"이라고 말했습니다. 즉, 새로운 무언가를 만든 게 아니라 자신이 이미 반복적으로 쓰던 코드를 묶어 하루 만에 판매 페이지를 올린 것입니다. 많은 개발자가 "이건 너무 단순해서 팔 수 없다"고 생각할 수준의 코드가 $199 원타임 결제 상품이 됐고 첫 달 $40,000을 벌었습니다.

**2. 빌드 인 퍼블릭 = 광고비 0원 배포 채널**

Marc Lou의 X 계정은 그 자체가 제품 발사대입니다. 매출 스크린샷, 실패 회고, 코드 짜는 영상을 꾸준히 올리면서 팔로워 250,000명을 확보했습니다. 새 제품을 출시하면 트윗 한 장이 Product Hunt 1위로 이어지고, 그게 다시 뉴스레터 구독자로 전환됩니다. 그의 말을 직접 인용하면: "Ideas are worthless without execution, and execution is worthless without market validation.(아이디어는 실행 없이 무가치하고, 실행은 시장 검증 없이 무가치하다.)" 그는 이 검증을 공개 트윗으로 실시간 진행합니다.

**3. TrustMRR — "문제 트윗 24시간 뒤 출시" 공식**

2024년 10월, Pieter Levels의 바이럴 트윗이 인디해커 커뮤니티에서 "매출 스크린샷 조작" 문제를 수면 위로 올렸습니다. Marc Lou는 그 트윗을 보고 24시간 안에 TrustMRR을 만들었습니다. Stripe API 키를 연결하면 검증된 MRR을 공개 페이지에 표시해 주는 사이트였습니다. 출시 다음 날 $20,378을 벌었고, 사이드바 광고 슬롯 가격을 $299 → $699 → $999 → $1,499로 올리면서 3일 만에 매진시켰습니다. 2025년 기준 TrustMRR 월 매출은 $31,400으로 그의 포트폴리오 중 1위입니다.

**4. 원타임 결제 + 포트폴리오 복리 모델**

Marc Lou의 제품 대부분은 구독이 아닌 원타임 결제($199~$299)입니다. 구독보다 초기 매출 피크가 높고, 유지 관리 부담이 낮습니다. 여기서 중요한 점은 그가 제품을 하나씩 쌓아가면서 서로 교차 홍보한다는 것입니다. ShipFast 구매자에게 CodeFast를 팔고, CodeFast 수강생에게 ShipFast를 권합니다. 2026년 1월 포트폴리오 현황: TrustMRR $31,400 / CodeFast $23,500 / DataFast $17,500 / ShipFast $17,200, 그 외 소규모 제품들이 나머지를 채웁니다.

**5. "실패의 복리"에서 시작한 이야기**

2017년 Marc Lou는 서울 공항에서 눈물을 흘렸습니다. 친구와 함께 창업한 AI 스타트업이 실패한 직후였습니다. 이후 4년간 여러 스타트업을 시도했지만 월 $3,000 이상으로 올라간 적이 없었습니다. 발리로 돌아왔을 때 통장에는 $20,000이 남아 있었습니다. 이 배수진 상황이 "1주일 안에 출시해서 검증"이라는 전략을 낳았습니다. 완벽함을 추구할 시간적·재정적 여유가 없었기 때문입니다.

## 와당탕/느린호밀 적용 포인트

**오늘 30분 (micro)**: 옵시디언에서 "미완성 아이디어 목록" 노트를 열고, 오늘 안에 남에게 보여줄 수 있는 가장 작은 버전 하나에 ✅ 표시 후 GitHub 이슈나 트위터 드래프트로 공개 선언하기. "이번 주 안에 만들겠습니다"라는 문장 하나면 충분합니다.

**이번 주 1-2시간 (mid)**: 와당탕 ERP 또는 느린호밀 비서앱에서 이미 반복적으로 사용하는 코드 블록(예: Claude API 호출 래퍼, 주문 상태 업데이트 함수)을 골라서 GitHub Public Gist나 간단한 npm 패키지로 묶기. Marc Lou식으로 가장 단순한 판매 랜딩 페이지(Notion 공개 페이지 + Stripe 결제 링크)를 연결하면 됩니다. 예를 들어 `hyein-daily`에서 자주 쓰는 Obsidian → Claude API 파이프라인을 모듈화해 "한국어 세컨드 브레인 보조 스크립트" 형태로 공개 GitHub 레포에 올리는 것이 출발점입니다.

```js
// 예시: 반복 사용 Claude API 호출을 한 줄 함수로 묶기
export async function askClaude(prompt, systemPrompt = '') {
  const msg = await anthropic.messages.create({
    model: 'claude-sonnet-4-6',
    max_tokens: 1024,
    system: systemPrompt,
    messages: [{ role: 'user', content: prompt }],
  });
  return msg.content[0].text;
}
```

**이번 달 실험 (macro)**: "와당탕 운영 자동화 레포트" 형식으로 월 1회 공개 스레드를 X나 LinkedIn에 작성합니다. 내용은 이번 달 Claude Code로 자동화한 것 1개, 절약한 시간(시간 단위), 구체적 코드 스니펫 1개입니다. 성공 기준: 스레드 임프레션 500회 이상 또는 GitHub 레포 스타 10개. 이것이 대표님의 "빌드 인 퍼블릭" 실험 시작점입니다.

## 한국 솔로 운영자 맥락에서 주의

**Product Hunt·영어권 배포 채널 의존 문제**: Marc Lou의 매출 폭발은 Product Hunt, X(영어 팔로워), Hacker News 등 영미권 플랫폼에서 나왔습니다. 한국어 1인 운영자가 동일한 전략을 쓰려면 플랫폼을 바꿔야 합니다. 네이버 블로그·카카오 오픈채팅·링크드인 한국 IT 커뮤니티가 대안이지만, 인디해커 커뮤니티만큼 "결제 전환률이 높은 개발자 독자"를 모으기 어렵습니다. $199 원타임 결제 보일러플레이트 모델은 영어권 SaaS 시장을 전제로 합니다.

**250K 팔로워가 없으면 론칭 효과 반감**: TrustMRR이 24시간 만에 $20,000을 번 것은 Marc Lou가 이미 수십만 팔로워를 가진 상태에서 한 일입니다. 팔로워 없이 동일한 1주일 출시 전략을 쓰면 매출 없는 실험 반복이 될 수 있습니다. 대표님은 먼저 특정 커뮤니티(와당탕 고객, 느린호밀 독자, Obsidian 한국 유저 그룹) 안에서 신뢰를 쌓은 뒤 공개 배포하는 순서가 적합합니다.

## 강의 메모 후보 (Pain/숫자/삽질/훅)

- **Pain**: 2017년 서울 공항에서 Marc Lou는 눈물을 흘렸습니다. AI 스타트업을 공동 창업하러 한국까지 날아왔지만 아무것도 안 되고 돌아가는 길이었습니다. 그때 그는 '더 오래, 더 완벽하게 만들어야 한다'는 믿음을 버리기로 결심했습니다.

- **숫자**: 2023년 9월 ShipFast 론칭 후 4월 2024년에는 월 $133,000까지 치솟았습니다. 27개 프로젝트에서 긁어모은 코드를 하루 만에 포장해 $199에 판 결과입니다. 같은 코드를 "내가 쓰는 내부 도구"로 뒀다면 0원이었을 자산입니다.

- **삽질**: TrustMRR을 처음 올렸을 때 수익화 계획이 없었습니다. 사이드바에 광고 슬롯을 급히 추가하면서 첫 슬롯을 $299에 팔았는데, 30초 만에 팔렸습니다. 그제야 가격을 올리기 시작해 $1,499까지 갔습니다. 처음부터 수익화 전략이 있었다면 더 많이 벌었을 수도 있지만, 수익화 없이 일단 올렸기에 24시간 안에 출시가 가능했습니다.

- **훅**: "세상에서 가장 잘 팔리는 개발자 도구 중 하나는 그 개발자가 이미 만들어 놓은 코드를 정리한 것입니다. 여러분도 지금 그 코드를 갖고 있습니다. 아직 판매 페이지가 없을 뿐이에요."
